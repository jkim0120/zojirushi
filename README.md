# Zojirushi
Zojirushi is an app that connects my Zojirushi rice cooker to the internet, allowing me to cook my rice remotely.

Deployed at: [zojirushi.herokuapp.com](http://zojirushi.herokuapp.com)

## Hardware Used
* Spark Core
* PowerSwitch Tail II
* Zojirushi Rice Cooker

## Software Used
* Sinatra
